Digital Student ID Management System

  This project is a full-stack web application designed to modernize campus identity management by replacing traditional physical ID cards with a secure and scalable digital solution. It enables institutions to efficiently manage student records while providing students with easy access to their digital identity and academic  services.

Technologies Used

  Frontend: HTML, CSS, JavaScript
  
  Backend: PHP
  
  Database: MySQL
  
  Development Tools: XAMPP Server
  
 Key Features
 
   Digital ID Generation – Automatically generates unique digital IDs for students
   
   Secure Authentication System – Login and access control for students and admin
   
   Admin Dashboard – Manage student data (add, update, delete records)
   
   Bonafide Certificate Request – Students can request bonafide certificates digitally through the system
   
   Real-Time Data Handling – Efficient storage and retrieval using MySQL
   
   User-Friendly Interface – Clean and simple UI for better usability
